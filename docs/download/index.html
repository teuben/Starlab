CVS Access

Alternatively, you can download the most current version from our CVS server by using the command 
    cvs -d :pserver:anonymous@cvs.astro.umd.edu:/home/cvsroot checkout starlab
to copy the entire package, or 
    cvs -d :pserver:anonymous@cvs.astro.umd.edu:/home/cvsroot update
to update a previous version.

Note: you won't be able to update a version extracted from any of the tarballs listed above. Update only works with a CVS release.
